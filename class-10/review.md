- Loops how and when to use

```javascript
// While loop: We use when we don't know how many times we need to run.
// For loop runs a certain number of times

let num = 0;
while(num == 0){
    prompt("Enter a number");
}

for(i = 0; i < 10; i++){
    // This will run 10 times
    console.log(i);
}
```
